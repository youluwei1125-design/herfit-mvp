# HerFit AI 逻辑说明

## 1. AI 功能概述

HerFit 是一款面向女性健身用户的 AI 日常训练决策助手，核心目标是根据用户的**生理周期阶段、身体状态、训练目标和当日训练场景**，生成更适合当天执行的训练建议。

HerFit 的 AI 逻辑不是简单生成一套通用健身计划，而是帮助用户回答一个更具体的问题：

> 结合我今天的身体状态和周期阶段，我今天适合做什么训练？

因此，HerFit 的核心价值不是提供更多训练内容，而是提供更个性化、更安全、更符合身体状态的训练决策支持。

---

## 2. 核心产品逻辑

HerFit 的推荐逻辑基于三个核心维度：

> **周期阶段 × 身体状态 × 训练场景**

这三个维度共同决定当天训练建议的方向。

### 2.1 周期阶段

不同周期阶段会影响用户的能量水平、身体感受和训练适应性，因此 HerFit 会根据周期阶段调整推荐强度和训练类型。

主要周期阶段包括：

- 月经期
- 卵泡期
- 排卵期
- 黄体期
- 未知 / 周期不规律

### 2.2 身体状态

用户当天的身体和情绪状态会直接影响训练是否适合进行，以及适合进行什么强度的训练。

常见身体状态包括：

- 疲劳
- 痛经
- 腹胀
- 情绪低落
- 压力大
- 睡眠差
- 头痛
- 精力充沛
- 状态正常

### 2.3 训练场景

HerFit 也需要结合用户当天的实际训练条件，避免生成不现实的训练建议。

主要训练场景包括：

- 训练目标：减脂、塑形、力量、拉伸、恢复
- 可用时间：10 / 20 / 30 / 45 分钟
- 偏好强度：低 / 中 / 高强度
- 可用器械：徒手、瑜伽垫、哑铃、健身房器械
- 训练地点：居家 / 健身房 / 户外

## 3. 用户输入结构

HerFit 的 AI 推荐系统需要接收结构化的用户输入。

示例输入：

```json
{
  "cycle_phase": "黄体期",
  "symptoms": ["疲劳", "腹胀"],
  "energy_level": "低",
  "fitness_goal": "塑形",
  "available_time": "20分钟",
  "preferred_intensity": "中等",
  "equipment": ["瑜伽垫", "哑铃"],
  "workout_location": "居家"
}
```

通过这些输入，HerFit 可以生成比普通训练计划更具体、更符合当天状态的推荐结果。

---

## 4. 推荐逻辑框架

HerFit 的推荐逻辑采用两层结构：

> **规则型安全过滤 + AI 生成式训练推荐**

也就是说，系统不会直接让 AI 随机生成训练内容，而是先根据用户状态进行安全判断，再让 AI 在安全边界内生成个性化训练建议。

整体流程可以理解为：

1. 用户输入周期阶段、身体状态和训练需求。
2. 系统先进行安全过滤，判断是否需要降低训练强度或避免某些动作。
3. 系统根据周期阶段调整训练方向。
4. AI 生成具体训练建议。
5. 输出推荐理由、安全提醒和替代方案。

---

## 5. 规则型安全过滤

安全过滤的作用是避免系统推荐不适合用户当前状态的训练内容。

如果用户报告明显不适，系统应优先降低训练强度，避免高冲击、高压迫或过度消耗的训练。

| 用户状态 | 推荐调整 |
|---|---|
| 月经期 + 痛经 | 避免 HIIT、跳跃类动作、高强度核心训练 |
| 严重疲劳 | 推荐恢复、拉伸、低强度活动 |
| 睡眠差 + 精力低 | 避免高强度力量训练或高强度有氧 |
| 头痛 / 头晕 | 推荐休息或非常轻度的舒缓活动 |
| 黄体期 + 腹胀 | 避免高冲击跳跃和强腹压训练 |
| 压力大 / 情绪低落 | 推荐瑜伽、拉伸、散步等温和训练 |

这部分逻辑的目标不是进行医学判断，而是让训练建议更保守、更安全、更贴合用户当天身体状态。

## 6. 基于周期阶段的训练逻辑

### 6.1 月经期

产品逻辑：

- 优先考虑舒适、恢复和低强度活动。
- 如果用户有痛经、疲劳或不适，应避免高强度训练。
- 推荐拉伸、轻瑜伽、散步、舒缓活动。
- 不鼓励用户在明显不适时强行训练。

推荐方向：

> 今天的训练应以恢复和舒缓为主，而不是追求运动表现。

### 6.2 卵泡期

产品逻辑：

- 月经结束后，部分用户精力和训练状态可能提升。
- 如果用户没有明显不适，可以推荐中高强度训练。
- 适合力量训练、渐进超负荷训练、有氧训练或技能训练。

推荐方向：

> 如果用户状态良好，可以安排更积极的力量训练或中高强度训练。

### 6.3 排卵期

产品逻辑：

- 部分用户在排卵期可能精力较好。
- 可以推荐全身力量训练、中高强度训练或更具挑战性的训练内容。
- 仍需结合用户当天症状和强度偏好，不能只根据周期阶段推荐高强度训练。

推荐方向：

> 如果用户精力较好，可以推荐更有挑战性的训练；如果有不适，则优先降低强度。

### 6.4 黄体期

产品逻辑：

- 黄体期可能伴随疲劳、腹胀、情绪波动、睡眠变差等情况。
- 推荐中低强度、低冲击、可调整的训练内容。
- 如果用户报告疲劳、腹胀或睡眠差，应避免高强度 HIIT 和高冲击训练。

推荐方向：

> 今天的训练应在活动和恢复之间取得平衡，重点是低冲击和可持续完成。

### 6.5 未知 / 周期不规律

产品逻辑：

- 如果用户不知道当前周期阶段，系统不应强行假设。
- 推荐逻辑应更多依赖用户当天的身体状态、精力水平和训练场景。
- 在信息不足时，推荐应更保守，并提供可调整选项。

推荐方向：

> 由于周期阶段未知，今天的建议主要基于用户当前身体状态和训练需求生成。

## 7. AI 输出结构

经过安全过滤和周期阶段判断后，AI 需要输出结构化训练建议。

推荐结果应包括：

1. 今日推荐训练类型
2. 推荐训练强度
3. 建议训练时长
4. 具体训练动作
5. 推荐理由
6. 安全提醒
7. 替代方案

示例输出：

```json
{
  "workout_type": "低冲击塑形 + 舒缓活动",
  "recommended_intensity": "低到中等",
  "duration": "20分钟",
  "exercise_plan": [
    {
      "exercise": "臀桥",
      "sets": "3组",
      "duration_or_reps": "每组12次"
    },
    {
      "exercise": "徒手深蹲",
      "sets": "3组",
      "duration_or_reps": "每组10次"
    },
    {
      "exercise": "猫牛式拉伸",
      "sets": "2组",
      "duration_or_reps": "每组45秒"
    }
  ],
  "reason": "用户处于黄体期，并反馈疲劳和腹胀，因此推荐低冲击训练，避免高强度有氧和强腹压动作。",
  "safety_note": "如果训练过程中不适感增强，应降低强度或停止训练。本建议不构成医疗建议。",
  "alternative": "如果今天精力很低，可以改为10分钟轻柔拉伸。"
}
```

- 力量训练

- 适合全身或下肢训练

> 推荐 45 分钟哑铃力量训练，重点放在复合动作和力量提升。

- 提供更轻松的替代方案

输出总结：

## 9. Prompt 设计逻辑

HerFit 的 AI prompt 需要明确以下内容：

- AI 的角色
- 用户输入字段
- 安全边界
- 输出格式
- 回复语气

示例 Prompt：

```text
你是 HerFit，一款面向女性用户的 AI 训练建议助手。

你的任务是根据用户的生理周期阶段、身体状态、训练目标、可用时间、偏好强度、可用器械和训练地点，生成安全、实用、周期友好的每日训练建议。

你不能提供医学诊断，也不能声称可以治疗任何经期或健康问题。

如果用户报告严重不适、头晕、明显疼痛或极低精力，应优先推荐休息、轻柔拉伸或低强度恢复活动。

当用户报告痛经、严重疲劳、腹胀或睡眠差时，应避免推荐高冲击、高强度或强腹压训练。

请按照以下结构输出：
1. 今日推荐训练类型
2. 推荐训练强度
3. 建议训练时长
4. 具体训练动作
5. 为什么适合今天
6. 安全提醒
7. 更轻松的替代方案
```

## 10. 产品边界

HerFit 不是医疗产品，也不能替代医生、康复师或专业教练的建议。

AI 不应该：

- 诊断疾病
- 声称可以治疗经期症状
- 鼓励用户忍痛训练
- 忽视用户反馈的不适
- 推荐极端燃脂或不安全训练
- 在用户头晕、严重痛经或异常症状时继续推荐高强度训练

AI 应该：

- 在不确定时给出更保守的建议
- 解释为什么推荐某种训练
- 提供低强度替代方案
- 在用户明显不适时建议休息
- 提醒用户如果症状严重或异常，应寻求专业建议

## 11. 为什么 HerFit 需要 AI

HerFit 使用 AI 的原因是：女性训练决策本身是一个动态、个性化、多变量的问题。

普通静态训练计划很难同时适配：

- 不同生理周期阶段
- 每天变化的身体状态
- 不同训练目标
- 不同时间和器械限制
- 不同强度偏好
- 用户临时出现的不适症状

AI 的价值在于，它可以把这些分散的输入整合起来，生成更贴合当天状态的训练建议，并解释推荐原因。

因此，HerFit 中 AI 的价值不只是“生成训练内容”，而是：

> 提供基于上下文的训练决策支持。

## 12. 未来 AI 迭代方向

后续版本可以进一步增强 HerFit 的 AI 逻辑，包括：

- 增加症状严重程度分级
- 加入训练后的用户反馈
- 记录用户历史训练表现
- 结合周期历史和状态变化进行个性化调整
- 接入可穿戴设备数据，例如睡眠、心率、活动量
- 增加推荐解释透明度
- 增加推荐置信度评分

未来示例逻辑：

```json
{
  "user_feedback": "今天的训练有点太累",
  "future_adjustment": "在类似的黄体期 + 低精力状态下，下次自动降低推荐强度"
}
```

这样 HerFit 可以从一次性的训练推荐工具，逐步发展为更个性化的 AI 女性健身助手。
